# 邮件聚合系统 - 需求文档

## 目标

个人使用的 Windows 桌面邮件客户端，将多个邮箱账户聚合到一个应用中统一管理。

## 功能需求

### 账户管理

- 添加邮箱账户：输入邮箱地址，自动发现服务器配置（AutoDiscovery）
- 自动发现失败时，支持手动填写 IMAP/SMTP 服务器信息
- OAuth 登录：Gmail、Microsoft、Yahoo、AOL、Fastmail
- 密码/授权码登录：163、QQ、iCloud 等
- 每个账户可独立配置 SOCKS5 代理
- 删除账户
- 编辑账户配置

### 收邮件

- 拉取收件箱邮件列表（发件人、主题、时间、摘要）
- 查看邮件正文（纯文本 + HTML）
- 查看/下载附件
- 多账户统一收件箱视图
- 按账户筛选邮件
- 标记已读/未读
- IMAP IDLE 实时接收新邮件通知

### 发邮件

- 写新邮件（收件人、抄送、密抄、主题、正文）
- 回复/回复全部/转发
- 添加附件
- 选择发件账户

### 文件夹管理

- 显示 IMAP 文件夹列表（收件箱、已发送、草稿、垃圾邮件等）
- 移动邮件到指定文件夹
- 删除邮件（移动到垃圾箱）

### 本地缓存

- 邮件缓存至本地 SQLite，离线可查看已缓存的邮件
- 账户配置与凭据加密存储至本地

## 非功能需求

- 仅支持 Windows 平台
- 纯客户端架构，不依赖后端服务器
- 应用级代理，不影响系统网络
